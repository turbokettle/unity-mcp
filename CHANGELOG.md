# Changelog

## [1.0.0] - 2025-02-06

### Added
- Initial release as UPM package and npm MCP server
- Dynamic tool system with automatic discovery via `IMcpTool` interface
- Built-in tools: `read_console_logs`, `execute_menu_item`, `get_project_info`
- `wait_for_editor_ready` tool for domain reload handling
- Auto-reconnection on Unity domain reload
- WindowWaker for keeping minimized Unity responsive
